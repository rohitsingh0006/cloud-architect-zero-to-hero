# Day 2 : Well Architected Framework (WAF)




|  Analogy                                  | Cloud  |
|-------------------------------------------|--------|
|✔️**Cost Optimization:** Are you overspending ? <br><mark>- reduce food waste, optimize staffing, buy in bulk </mark></br> | **Cost Optimization:** Lower cloud spends <br><mark>- Use Azure cost mgmt, reserve instances, right-sized VMs</mark></br> |
|✔️**Security:** Is the restaurant secure ? <br><mark>- Install CCTV camera, lock doors, background checks on staff</mark></br>|**Security:** Protect data & Access <br><mark>- Use zero trust model, Azure firewall, defender for cloud, MFA</mark></br> |
|✔️**Reliability:** Can you serve during peak hours or power cuts ? <br><mark>- backup generator, extra staff on weekends</mark></br>|**Reliability:** Ensure uptime and recovery <br><mark>- Use availability zones, geo redundant storage, backups, auto-scaling</mark></br> |
|✔️**Operational Excellence:** Are operations smooth and effecient ? <br><mark>- Train staff, use a POS system, daily checklist</mark></br>|**Operational Excellence:** Automate and monitor <br><mark>- Implement CI/CD pipelines, use Azure monitor, set alerts</mark></br> |
|✔️**Performance Effeciency:** Can you serve faster and better ?<br><mark>- automate order taking, pre-prep popular items, improve kitchen layout</mark></br>|**Performance Effeciency:** Deliver optimal performance <br><mark>- Use Azure front door, CDN, caching, right SKUs, autoscaling logic</mark></br> |
