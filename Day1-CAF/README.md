# Day 1 : Cloud Adoption Framework (CAF)




|  Analogy                                  | Cloud  |
|-------------------------------------------|--------|
|✔️**Strategy:** Why do I need car? <br><mark>- for daily commute to work</mark> </br> |**Strategy:** Why cloud ? <br><mark>- Need to scale app for global users</mark></br>|
|✔️**Plan:** 	What type, budget, features ? <br><mark>- Sedan under 1m with good mileage</mark></br> |**Plan:** Which workload, how much budget and skill required <br><mark>- identify apps to migrate, estimate cost, train team on Azure</mark></br>|
|✔️**Ready &/mark Govern:** Get liscence, research rules, check parking space, prepare for exam <br><mark>- prepare documentation and ensure legal compliance</mark></br> |**Ready &/mark Govern:** Prepare environment <br><mark>- create landing zone with governance policies(naming, tagging, cost limits)</mark></br>|
|✔️**Adopt:** Buy the car and start driving <br><mark>- purchase, register and being using it</mark> </br> |**Adopt:** Move to Azure <br><mark>- Use Azure migrate or build native apps</mark></br>|
|✔️**Secure:** Insurance, anti-theft system, front/back camera <br><mark>- protect it and yourself</mark></br>  |**Secure:** Enable security <br><mark>- Defender for cloud, RBAC, NSG, identity protection</mark></br>|
|✔️**Manage:** Regular servicing and fueling tank <br><mark>- maintain performance and track expenses</mark></br>  |**Manage:** Monitor and optimize <br><mark>- Azure monitor, Update management, cost tracking</mark></br>|
|✔️**Organize:** assign who can drive it <br><mark>- define users and organize usage pattern</mark></br> |**Organize:** Define roles and structure <br><mark>- setup subscription , management groups, RBAC</mark></br>|
